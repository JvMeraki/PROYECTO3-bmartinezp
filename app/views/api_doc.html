Documentación de la API de la Heladería

Esta API permite gestionar productos e ingredientes de la heladería.

Endpoints disponibles

Productos

  • Consultar todos los productos: GET /productos_json
  • Consultar un producto por ID: GET /producto/<id>
  • Consultar un producto por nombre: GET /producto/nombre/<nombre>
  • Consultar calorías de un producto: GET /producto/<id>/calorias
  • Consultar rentabilidad de un producto: GET /producto/<id>/rentabilidad
  • Consultar costo de producción de un producto: GET /producto/<id>/costo
  • Vender un producto: POST /producto/<id>/vender
  • Reabastecer un producto: POST /producto/<id>/reabastecer
  • Renovar inventario de un producto: POST /producto/<id>/renovar

Ingredientes

  • Consultar todos los ingredientes: GET /ingredientes_json
  • Consultar un ingrediente por ID: GET /ingrediente/<id>
  • Consultar un ingrediente por nombre: GET /ingrediente/nombre/<nombre>
  • Verificar si un ingrediente es saludable: GET /ingrediente/<id>/saludable
